# MOID

The Massive Object Image Database (MOID) is a newly curated dataset containing 24,592 images across 1,468 object concepts and 13 broad categories. It provides an extensive taxonomy of consistently scaled object images, each cleanly isolated against a uniform white background.

The [MOID website](https://lpez122.github.io/MOID/) offers a curated preview of the collection. For research access, contact [luke.pezanko@utdallas.edu](mailto:luke.pezanko@utdallas.edu).
